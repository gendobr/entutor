ETU

------------------------------
Термины
------------------------------

Гость - любой посетитель сайта, который не вошел с именем/паролем или через сторонний сервис

Студент - посетитель сайта, который вошел с именем/паролем или через сторонний сервис. Студент имеет право записываться на курсы и учиться.

Методист - человек, который имеет право редактировать Курсы, Уроки и Упражнения

Администратор - человек, который имеет максимальный набор прав

Пользователь - это Студент или Методист или Администратор

Уровень курса - (например, начальный, средний, сложный)

Курс - последовательность Уроков

Урок - последовательность Упражнений
	Один урок может встречаться в разных курсах

Упражнение - минимальный элемент, который показывает студенту задания 
	и иногда принимает ответы.
	Одно упражнение может встречаться в разных уроках.


------------------------------
Варианты использования
------------------------------


Гость регистрируется на сайте:
	вводит имя пользователя (логин), пароль, электронную почту
	после регистрации пользователю на почту отправляют ссылку для подтверждения регистрации
	новый пользователь получает права Студента


Гость подверждает свою регистрацию
	открывает почту и нажимает на ссылку


Гость запрашивает повторное подверждение регистрации
	отправляет на сайт свою электронную почту
	открывает почту и нажимает на ссылку


Гость входит через социальную сеть
	facebook
	vk
	twitter
	одноклассники
	google
	yandex


Гость просматривает описание сервиса
	Это связанный сайт с обычной системой управления: страницы, новости, форум и т.д.


Пользователь изменяет свои данные
	Имя, фамилию, пароль, электронную почту


Студент просматривает список доступных Курсов
	У курсов есть Уровень (например, начальный, средний, сложный)
	В списке некоторые Курсы уже доступны студенту,
	некоторые доступны бесплатно
	некоторые доступны после оплаты
	У каждого Курса есть в списке название, иконка и краткое описание


Студент просматривает список своих курсов
	У студента есть список курсов, к которым он получил доступ на N дней
	Возле каждого курса стоит показатель прогресса ( 0 - 100 % )


Студент просмотривает информацию о курсе
	Студент видит название, описание курса, процент пройденных уроков, список уроков курса.
	Если урок пройден, то возле него стоит флажок
	Если курс доступен студенту
		Если урок пройден, то он доступен для повторного прохождения
		1-й Урок всегда доступен
		Урок, следующий после пройденного доступен.
		Все остальные уроки недоступны (чтобы студент не пропускал упражнения)
    Если курс недоступен, то есть кнопка "Записаться"
    	- Если курс бесплатный, то запись происходит после нажатия кнопки
    	- Если курс платный, то студента отправляют в Интеркассу для оплаты,
    и после оплаты курс становится доступным, о чем студент извещается по электронной почте
    Доступ к курсу ограничен по времени (например, 2 месяца от момента записи)


Студент просматривает урок (только если курс доступен студенту)
	В начале показан курс и прогресс в курсе
	Урок состоит из упражнений и список упражнений виден студенту (это просто номера упражнений)
	Пройденные упражнения доступны для повторного прохождения, из нескольких результатов сохраняется лучший/средний.
	1-е упражнение доступно
	Упражнения, следующее после пройденного, доступно.
	Остальные упражнения недоступны.


Студент просматривает упражнение
	В начале показан курс и прогресс в курсе.
	Потом - название урока и прогресс в уроке.
	Упражнения делятся на требующие ответа и не требующие ответа.
	Если упражнение не требует ответа, то его просмотр на протяжении 10 секунд устанавливает отметку "пройдено".
	Если упражнение требует ответа, то Студент выполняет задание и получает отметку "пройдено" в случае успешного ответа.
	У каждого упражнения есть счётчик времени, который показывает время, затраченное студентом на упражнение.
	У каждого упражнения есть счётчик правильных и неправильных попыток



Методист просматривает список курсов
	Страница показывает таблицу со списком курсов с колонками
		- меню
		- код курса
		- признак Скрытый/Видимый
		- уровень курса
	   	- количество студентов, у которых есть доступ к курсу
		- название курса
		- стоимость
		- длительность
	Нажатие на меню показывает список доступных операций


Методист создаёт информацию о курсе
Методист изменяет информацию о курсе
	Форма редактирования, в которой можно задать
	- название
	- иконку
	- краткое описание
	- признак Скрытый/Видимый

Методист удаляет курс

Методист просматривает список уроков курса
	В начале страницы показано название курса и меню Курс"
	Порядок уроков можно изменять перетаскиванием строк
	Потом идёт список уроков в каждой строке которого
		- Кнопка меню "Урок"
		- Код урока
		- номер по порядку
		- Название урока


Методист создаёт урок
Методист изменяет информацию об уроке
	Форма редактирования содержит
	- меню "Курс"
	- меню "Урок"
	- название Урока
	- Иконка урока
	- Курс, к которому Урок относится
	- признак Скрытый/Видимый
	- краткое описание урока


Методист изменяет список упражнений урока
	Список упражнений содержит 
	- меню "Упражнение"
	- код упражнения
	- номер по порядку
	- служебное название упражнения
	Упражнения добавляются из общей базы упражнений


Методист просматривает базу упражнений
	Список упражнений содержит 
	- меню "Упражнение"
	- код упражнения
	- количество уроков, в которых есть это упражнение
	- признак Скрытый/Видимый
	- служебное название упражнения


Методист изменяет упражнение
	Форма редактирования содержит
	- служебное название упражнения
	- Описание упражнения для Студента
	- Программный код упражнения
	- Подсказка Студенту (или даже правильный ответ)
	- Позже надо добавить редактор Упражнения
	- кнопка "Предварительный просмотр"


Методист управляет файлами сайта
	Простой менеджер файлов:
	- Список файлов и директорий
	- загрузка, переименование, перемещение, удаление файлов


Администратор управляет пользователями
 	- список
 	- изменить
 	- удалить
 	- создать


Администратор управляет уровнями курсов
 	- список
 	- изменить
 	- удалить
 	- создать


------------------------------------------- 
Сервер проверки произношения
------------------------------------------- 

Сервер получает
- фразу (фрагмент текста)
- цифровую подпись (для проверки прав доступа)
- звукозапись (до 30 секунд)

Возвращает
- общую оценку фразы (число от 0 до 1)
- оценку для каждого слова (число от 0 до 1)



-------------------------------------------
Структура упражнений
-------------------------------------------

Упражнение содержит
	Презентацию (некий текст, который объясняет, что должен сделать студент)
	Форму для ввода ответов
	Кнопки "Далее", "Начать заново", "Проверить"
	Можно включить автоматический старт проверки ответов

Элемент card
	содержит другие элементы
	может выстроить их по вертикали или по горизонтали

Элемент html
	содержит текст, картинки и др. Всё, что может быть показано на веб-странице.
	ответы студента не принимает

Элемент text
	Показывает текстовое поле для ввода ответа
	Умеет проверять правильность ответа

Элемент radio
	Показывает набор радиокнопок (*)
	Позволяет студенту выбрать один вариант ответа из нескольких
	Умеет проверять правильность ответа

Элемент checkbox
	Показывает флажок (checkbox)
	Позволяет студенту поставить или убрать отметку
	Умеет проверять правильность ответа

Элемент counter
	Показывает фрагмент текста, который можно перетаскивать в специально подготовленные места (см. элемент dropzone)
	ответы студента не принимает

Элемент dropzone
	Показывает пустой прямоугольник, в который можно перетащить специально подготовленный ответ (см. элемент counter)
	Умеет проверять правильность ответа

Элемент audio
	Принимает устный ответ с микрофона, и оправляет его на проверку серверу.
	Умеет проверять правильность ответа
	Показывает индикатор громкости, кнопки "Начать запись" и "Завершить запись"

Элемент sound
	показывает кнопку "Проиграть звук"
	ответы студента не принимает

Элемент video	
	показывает видеозапись
	ответы студента не принимает
