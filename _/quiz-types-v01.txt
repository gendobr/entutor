СRС

------------
Задание:
------------
- Поясняющий текст (что-то типа справки)
- Условие 
  - форматированный текст
  - рисунки
  - звук
  - видео
  - звук с субтитрами
  - видео с субтитрами
  - дырка
    - фишка
- Приемник ответов
  - карточка
  - кнопка "Проверить"
- Кнопка "Далее" (активна, если нет приемника ответов или проверка показала правильность ответов)


------------
Карточка
------------
getValue() :  string[]
test() : false|true|null - результат проверки ответов
hint() : html - подсказка, включается после нажатия кнопки "проверить"
parent : Карточка
children
  - html - показ информации
  - карточка
  - текстовое_поле
  - звукозапись_с_микрофона
  - место_для_фишки
  - радиокнопки (radiobutton)
  - флажок (checkbox) extends element
displaySuccess() - поздравляет с успехом
displayError() - показывает сообщение об ошибке
activate() - активировать карточку
deactivate() - деактивировать карточку
onTest() - срабатывает после очередной проверки ответов и, 
           в зависимости от pre-condition,
           показывает или скрывает карточку
pre-condition = true | previous-cards-correct - условия видимости

- иногда содержит шаблоны правильных ответов, если для оценки нужно учитывать комбинацию
- содержит размеры (ширина), высота регулируется по содержимому

------------
фишка
------------
Блок текста, который можно перетаскивать и ронять в специально отведённые места
Чаще всего это слово или символ.
Фишка помнит свою дырку и умеет в неё возвращаться


------------
дырка
------------
Место в Условии, где стоит фишка


------------
element
------------
id: string
getValue() : string[]
test() : false|true|null - результат проверки ответов
hint() : html - подсказка, включается после нажатия кнопки "проверить"
parent : Карточка
displaySuccess()
displayError()
draw()


------------
текстовое_поле  extends element
------------
- рисует текстовое поле длины X для ввода строки
- иногда содержит шаблон правильного ответа
- если есть шаблон правильного ответа, то метод test() умеет сравнивать его с ответом



------------
место_для_фишки extends element
------------
- рисует пустой прямоугольник ширины Х
- иногда содержит фишку
- может отправить фишку на исходную точку, если ответ неправильный
- иногда содержит шаблон правильного ответа
- если есть шаблон правильного ответа, то метод test() умеет сравнивать его с содержанием фишки



------------
радиокнопки (radiobutton) extends element
------------
- содержит все варианты_ответов
- содержит номер правильного варианта ответа


------------
вариант_ответа
------------
- содержит форматированный текст, рисунки, звукозаписи, видео - всё, что можно вставить в html



------------
флажок (checkbox) extends element
------------
- содержит один вариант_ответа
- содержит состояние, которое надо считать правильным ответом
  (false|true|null)


------------
звукозапись_с_микрофона extends element
------------
- идентификатор элемента - это текстовый 
- рисует кнопки "Начать запись", "Завершить запись", "Воспроизвести запись"
- при нажатии на кнопку "Начать запись" начинает звукозапись
- при повторном нажатии на кнопку "Завершить запись" завершает звукозапись и отправляет звук серверу,
- звук отправляется серверу вместе с идентификатором элемента
 (так будут собираться данные для дополнительного обучения программы)
  и дополнительными параметрами: количество слов, мужчина или женщина и т.д.
- сервер возвращает результат проверки - массив пар (слово, оценка) + общая оценка,
- при нажатии на кнопку "Воспроизвести запись" проигрывает звукозапись
